Welcome to the NBA Backend API Documentation

This documentation provides detailed information about the NBA Backend API. The API allows you to access various player statistics in different formats. Explore the endpoints below to retrieve the data you need.

Endpoints

Here are the available endpoints for retrieving player statistics:

/players

Returns a list of all players in the league in alphabetical order by last name.

Usage:

GET /players

Example Usage (JavaScript):

// Make a GET request to retrieve player data
fetch('/players')
  .then(response => response.json())
  .then(data => {
    // Process the response data
    console.log(data.players); // Print the list of players
    // Your code to handle the data goes here
  })
  .catch(error => {
    // Handle any errors that occurred during the request
    console.error('Error:', error);
  });

Example Response:

{
  "players": [
    {
      "id": 1,
      "firstName": "LeBron",
      "lastName": "James",
      "team": "Los Angeles Lakers",
      "position": "Forward",
      "jerseyNumber": 6
    },
    {
      "id": 2,
      "firstName": "Kevin",
      "lastName": "Durant",
      "team": "Brooklyn Nets",
      "position": "Forward",
      "jerseyNumber": 7
    },
    ...
  ]
}
